# Goals

The general goal is to generate a Jupyter notebook that generate a figure to included in a scientific paper. The idea is to illustrate how the EP (additive Epsilon, ε+) and HV (Hypervolume) quality indicators change when using estimated reference fronts. 

The figure must include:

1. A line with the data of the reference front of problem RE22, located in "resources/referenceFronts/RE22.csv".
2. A scatter plot with the approximated reference front of problem RE22, located in "resources/estimatedReferenceFronts/RE22.csv".   
3. The reference point used to compute the HV, which is derived from the estimated reference front.
4. A scatter plot with a front (Front A) composed of 20 points with a similar shape to the reference front of problem RE22. This front must be dominated by the reference front of problem RE22.
5. A scatter plot with a front (Front B) composed of 20 points with a similar shape to the reference front of problem RE22. This front must be partially dominated by the reference point.
6. A scatter plot with a front (Front C) composed of 20 points with a similar shape to the reference front of problem RE22. This front must dominate front B.

**Clarification**: Fronts A, B, and C are generated by displacing the reference front of RE22, maintaining a similar shape. Each front must contain only non-dominated solutions (i.e., each front is itself a Pareto front).

**Clarification**: The approximated reference front is obtained from the reference front by running the EstimatedReferenceFrontGenerator class with the offset percentage set to 0.30.

7. The HV and EP values of the three fronts must be computed and included in the figure. EP is computed with respect to the estimated reference front, and HV is computed using the reference point derived from the estimated reference front. To compute these quality indicators, you can use the implementations provided the jMetalPy project (local repository: https://github.com/jMetal/jMetalPy)

The resulting figure must show that the HV of Front A is zero, that the EPs of Fronts B and C are negative, and the EP of Front C is lower than the EP of Front B.

**Visualization**: Use Matplotlib (and Seaborn if needed) for generating the figure.

**Environment**: Use the Conda virtual environment named "Evolver".
