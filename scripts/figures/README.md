## Current Figures

### hv_comparison_convergence.png
- **Analysis**: Experiment A - HV Evolution Analysis
- **Description**: Comparison of hypervolume convergence between reference fronts and extreme points fronts
- **Generated**: 2026-03-17
- **Format**: PNG, 300 DPI (publication-ready)
- **Dimensions**: Optimized for two-column journal format

## Figure Contents

### Visual Elements
- **Grid layout**: 2 rows × 4 columns (problems × budgets)
- **Color scheme**: 
  - Blue (#1f77b4): Reference fronts (dense Pareto approximations)
  - Orange (#ff7f0e): Extreme points fronts (minimal reference)
- **Line style**: Solid lines, consistent across all subplots

### Data Represented
- **Problems**: RE3D (top row), RWA3D (bottom row)
- **Budgets**: 1000, 3000, 5000, 7000 base-level evaluations
- **Aggregation**: Median across 30 independent runs
- **Interpolation**: Forward-fill to 30 evenly-spaced checkpoints
