# Analysis Tools

This folder contains Python scripts and notebooks for analyzing and visualizing Evolver results.

## Structure

```
analysis/
├── scripts/
│   ├── visualization/     # Front and optimization progress visualization
│   │   ├── visualize_training_set_results.py  # Visualize fronts from TrainingSetRunner
│   │   └── plot_optimization_progress.py      # Plot meta-optimization progress
│   ├── ablation/          # Ablation analysis scripts (future)
│   ├── feature_importance/ # Feature importance analysis (future)
│   └── robustness/        # Robustness analysis scripts (future)
├── notebooks/             # Jupyter notebooks for interactive analysis
│   ├── Plot2D.ipynb       # 2D front visualization notebook
│   └── Plot3D.ipynb       # 3D front visualization notebook
├── sample_data/           # Sample data files for testing scripts
├── output/                # Generated plots and analysis results
└── requirements.txt       # Python dependencies
```

## Installation

```bash
cd analysis
pip install -r requirements.txt
```

## Scripts

### visualize_training_set_results.py

Visualizes Pareto fronts generated by `TrainingSetRunner`. Supports both 2D and 3D fronts.

```bash
# Basic usage (individual plots)
python scripts/visualization/visualize_training_set_results.py results/fronts/ZDT

# Grid view of all fronts
python scripts/visualization/visualize_training_set_results.py results/fronts/ZDT --grid

# Combined plot (all fronts overlaid)
python scripts/visualization/visualize_training_set_results.py results/fronts/ZDT --combined

# Interactive Plotly visualization (useful for 3D)
python scripts/visualization/visualize_training_set_results.py results/fronts/DTLZ3D --interactive

# Interactive grid view
python scripts/visualization/visualize_training_set_results.py results/fronts/DTLZ3D --interactive-grid
```

### plot_optimization_progress.py

Generates a static plot showing the progression of multi-objective optimization runs.

```bash
python scripts/visualization/plot_optimization_progress.py RESULTS/SPEA2/NSGAII/DTLZ3 200
```

## Notebooks

- **Plot3D.ipynb**: Interactive notebook for visualizing 3D Pareto fronts
