# Analysis Tools

This directory contains tools for visualizing and analyzing the data generated by the Java Evolver framework.

## Structure

*   **`notebooks/`**: Jupyter notebooks for interactive data exploration. Use these to prototype analyses and understand your data.
*   **`paper_artifacts/`**: Python scripts designed to generate publication-quality figures (PDF/EPS) and tables (LaTeX) directly from the results.
*   **`output/`**: Default destination for generated artifacts.

## Usage

1.  **Generate Data**: Run your Java meta-optimization experiment (e.g., `NSGAIIOptimizingNSGAIIForBenchmarkZDT`). This produces `CONFIGURATIONS.csv` and `INDICATORS.csv` in `RESULTS/`.
2.  **Generate Artifacts**: Run scripts from `paper_artifacts/` pointing to your results directory.

```bash
python paper_artifacts/generate_quality_plots.py --input ../RESULTS/NSGAII/ZDT
```

## Requirements

Install dependencies:
```bash
pip install pandas matplotlib seaborn scikit-learn jupyter
```
