# Analysis Tools

This folder contains Python scripts and notebooks for analyzing and visualizing Evolver results.

## Structure

```
analysis/
├── scripts/
│   ├── visualization/     # Front and optimization progress visualization
│   ├── ablation/          # Ablation analysis scripts
│   ├── feature_importance/ # Feature importance analysis
│   └── robustness/        # Robustness analysis scripts
├── notebooks/             # Jupyter notebooks for interactive analysis
├── output/                # Generated plots and analysis results
└── requirements.txt       # Python dependencies
```

## Installation

```bash
cd analysis
pip install -r requirements.txt
```

## Usage

See individual script documentation for usage instructions.
